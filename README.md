# AI Automation Platform — Landing Page

A premium, responsive landing page for an AI-driven data automation platform, built for Phase 1 of the hackathon.

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- TypeScript

## Features
- Dynamic multi-currency pricing matrix (INR/USD/EUR, Monthly/Annual toggle with 20% annual discount), driven by a multi-dimensional config object
- Responsive Bento Grid (desktop) → Accordion (mobile) with shared active-index state, enabling seamless context transfer across breakpoints
- Semantic HTML structure for SEO and accessibility
- Custom color palette and typography (JetBrains Mono + Inter) per provided design assets
